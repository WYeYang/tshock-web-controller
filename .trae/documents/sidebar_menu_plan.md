# 侧边栏菜单 + 视图切换计划

## 理解需求
侧边栏是导航菜单，点击菜单项切换主区域显示的内容，而不是把所有内容都放在侧边栏。

## 新的设计方案

### 1. 侧边栏菜单（Sidebar）
- 导航菜单项：
  - 命令助手（聊天界面）
  - 服务器状态
  - 帮助文档
  - 配置面板
- 高亮当前选中的菜单项
- 点击菜单项切换主视图

### 2. 主视图区域
根据选中的菜单项显示对应的内容：

#### 视图 1: 命令助手（默认）
- 聊天风格界面
- 执行命令
- 快捷命令按钮

#### 视图 2: 服务器状态
- 服务器详细信息
- 玩家列表
- 踢出/封禁功能

#### 视图 3: 帮助文档
- 配置说明
- 命令参考

#### 视图 4: 配置面板
- TShock 配置
- LLM 配置

### 3. 需要修改的文件
- `Sidebar.tsx` → 改为导航菜单
- `Dashboard.tsx` → 改为视图控制器
- 可能需要拆分子组件（可选）

## 实现步骤
1. 修改 Sidebar 为导航菜单，支持选中状态
2. 修改 Dashboard 为视图切换逻辑
3. 将各个功能拆分为独立视图（可复用现有代码）
4. 确保所有功能完整保留
