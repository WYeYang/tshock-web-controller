# 侧边栏 + 聊天界面布局重构计划

## 目标
将当前的标签页布局重构为：
- **侧边栏**：包含配置、服务器状态、帮助文档
- **主区域**：AI 聊天风格的命令控制台（改名为"命令助手"）

## 当前文件结构分析
- `Dashboard.tsx`：当前主界面，包含所有功能
- `ConfigPanel.tsx`：可折叠的配置面板
- `Navbar.tsx`：顶部导航栏
- 其他相关组件和工具

## 实现计划

### 1. 修改导航栏 (Navbar.tsx)
- 简化为只有品牌 logo，移除标签页导航
- 添加侧边栏切换按钮（移动端适配）

### 2. 创建侧边栏组件 (Sidebar.tsx)
将配置、服务器状态、帮助文档整合到可折叠/可展开的侧边栏：
- **配置面板**：折叠状态显示摘要，展开显示完整配置
- **服务器状态**：精简版状态显示（在线人数、服务器信息摘要）
- **帮助文档**：链接或快速访问

### 3. 重构主界面为聊天风格 (Dashboard.tsx)
将命令控制台改造为 AI 聊天界面风格：
- 改名"命令助手"或"TShock Bot"
- 聊天消息风格显示命令和响应
- 用户消息（发送的命令）在右侧
- 机器人消息（服务器响应）在左侧
- 输入框在底部，支持 Enter 发送
- 显示历史消息
- 快捷命令按钮
- 保持所有原有功能（踢人、封禁等）

### 4. 响应式设计
- 桌面端：固定侧边栏 + 主聊天区
- 移动端：可切换的侧边栏 + 全屏聊天区

### 5. 保持所有功能
- 踢人、封禁玩家
- 命令执行
- 服务器状态监控
- 配置管理
- Token 管理

## 实现步骤
1. 创建 Sidebar 组件，整合 ConfigPanel、精简版状态、帮助
2. 修改 Navbar 为简洁品牌栏
3. 重构 Dashboard 为聊天风格
4. 调整样式和布局
5. 测试和优化

## UI 风格保持
- 保持现有的 cyberpunk/霓虹灯风格
- 保持玻璃态卡片设计
- 保持渐变色和动效

## 注意事项
- 保持代码结构清晰
- 保持 TypeScript 类型安全
- 保持现有功能完整
- 保持响应式设计
