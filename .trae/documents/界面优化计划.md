# 界面优化计划

## 目标
优化玩家详情界面，改进buff图标和数据显示，简化操作界面

## 优化任务

### 1. Buff图标使用英文名（从buffs.json获取）
**文件**: `src/utils/terraria.ts`
- 修改 `getBuffIconUrl` 函数
- 从 `BUFF_DATA` 获取buff英文名称
- 生成格式: `https://terraria.wiki.gg/images/${英文名称.replace(/ /g, '_')}.png`
- 示例: "Obsidian Skin" → `https://terraria.wiki.gg/images/Obsidian_Skin.png`

### 2. BuffSlot组件显示中文描述
**文件**: `src/components/BuffSlot.tsx`
- 悬浮提示显示中文名称（已有 `getBuffName` 函数）
- 保留英文名称作为备选显示
- 优化tooltip显示格式

### 3. 删除清空按钮
**文件**: `src/components/PlayerDetailModal.tsx`
- 删除 "清空背包" 按钮
- 删除 "清空装备" 按钮
- 删除 "清空Buff" 按钮
- 保留其他操作按钮（踢出、封禁、禁言、修改用户组、给予物品）

### 4. 压缩基本信息版面
**文件**: `src/components/PlayerDetailModal.tsx`
- 基本信息区域使用更紧凑的布局
- 减少padding和margin (从 p-3 sm:p-4 改为 p-2 sm:p-3)
- 使用更小字体 (text-sm 改为 text-xs)
- 增加每行显示数量 (grid-cols-2 改为 grid-cols-3 或 grid-cols-4)
- 减小卡片间距 (gap-3 sm:gap-4 改为 gap-2 sm:gap-3)
- 减小标题字体 (text-base sm:text-lg 改为 text-sm sm:text-base)

## 实施步骤

1. 修改 `src/utils/terraria.ts` 从 `BUFF_DATA` 获取英文名称生成图标URL
2. 优化 `src/components/BuffSlot.tsx` tooltip显示中文
3. 更新 `src/components/PlayerDetailModal.tsx` 删除清空按钮
4. 优化基本信息区域的布局样式
5. 运行 `npm run build` 测试构建

## 预期效果

- Buff图标使用英文名正确加载到Wiki图片
- Buff悬浮提示显示中文描述
- 界面更简洁，移除清空操作按钮
- 基本信息区域更紧凑，节省空间占用
